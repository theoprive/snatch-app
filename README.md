# Snatch

Snatch is a location-based mobile platform designed to help users discover and book real-world experiences.

The app focuses on spontaneous discovery of activities and events through a social experience feed.

## Tech Stack
- React Native
- Expo
- TypeScript

## Key Features
- Experience discovery feed
- User profiles
- Messaging and connections
- AI-assisted product prototyping

## Project Status
MVP currently in development.
